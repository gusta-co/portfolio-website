Projeto

No aplicativo, feito para portfólio, é possível votar em um candidato e visualizar os resultados até o momento dos votos computados. O aplicativo móvel conta com um sistema de autenticação de 2 fatores, um formulário para cadastro do usuário, consome uma API própria para computar e gerar os resultados. Utiliza bancos de dados SQL (Postgres) e NoSQL (Firebase Database).
